# Weather Forecast Web App (Flask)

## Project Overview
Developed a weather forecasting web application using Flask that provides real-time weather updates based on user input location.

## Objective
To build a web application that fetches and displays live weather data using external APIs.

## Key Features
- Real-time weather data retrieval  
- Location-based weather search  
- Clean and responsive UI  
- API integration for live updates  

## Tech Stack
- **Backend:** Python (Flask)  
- **Frontend:** HTML, CSS  
- **API:** OpenWeatherMap API  

## Workflow
1. User enters location  
2. API request is sent to weather service  
3. Data is retrieved and processed  
4. Weather details are displayed on UI
